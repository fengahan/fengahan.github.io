PHP利用stack(栈)写了个简单都计算器

废话少说先上代码才是好汉,其核心部分为将人们平常用的中缀表达式(运算符在中间)转换为后缀表达式。计算的时候只需先

依次读出符号再读出该符号前两个数字即可,以下一个简单的示列:代码写的很丑 请务嘲笑😊

            if ((empty($oper_stack) || $char == "(") && $char != ')') {
                array_push($oper_stack, $char);

                    $pc=count($oper_stack);
